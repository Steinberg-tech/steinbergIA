## Mensagem

- Use quando o cliente parou de responder e precisa ser reengajado
- Tom simples, breve e não invasivo
- Se disponível, personalize com o nome do cliente no lugar de "Sr.(a)."
- Não repita este template mais de uma vez por semana para o mesmo cliente
